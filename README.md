# Contact Form
https://vyasmayuri2007-lang.github.io/Contact-Form/
This project is a functional and responsive Contact Form built using HTML, CSS, and JavaScript. It allows users to submit their name, email, subject, and message through a simple and user-friendly interface.

# Key Features

1.Clean and responsive UI for smooth user experience
2.JavaScript-based validation for accurate input (email format, required fields, etc.)
3.Clear input structure with labeled fields
4.Real-time success/error messages for better user feedback

# Purpose
To demonstrate core front-end development skills, including form design, validation, and responsive layout, while building a practical component used in modern websites.
